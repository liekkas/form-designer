# form-designer

## Build Setup

``` bash
# install dependencies
npm install form-designer

in main.js
```
import iView from 'iview'
import 'iview/dist/styles/iview.css';
import 'form-designer/dist/form-designer.min.css';

Vue.use(iView)
```
in App.vue
```
  import formView from 'form-designer';
```
